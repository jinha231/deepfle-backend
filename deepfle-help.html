1
STEP 1
계정 설정
🚀
가장 먼저 광고주(브랜드) 계정을 만들고, 함께 일할 팀원을 초대하세요. 역할에 따라 보이는 메뉴가 달라지니 처음에 잘 정해두면 좋아요.
🏢
① 계정 만들기
마스터 계정으로 로그인한 뒤 계정 관리 메뉴에서 "+ 계정 추가"를 눌러 광고주(클라이언트)별로 계정을 하나씩 만듭니다. 계정 1개 = 브랜드 1개라고 생각하면 쉬워요.
👥
② 팀원 초대하고 역할 정하기
마스터 — 모든 기능 사용 가능 (API 연동, 설정, 사용자 관리 등)
사용자 — 성과 조회 + 대부분의 데이터 입력·편집 가능
광고주 — 성과를 조회만 할 수 있어요 (수정 불가)
🔒
③ 참고: 마스터 전용 메뉴
자동 최적화 · 어트리뷰션 링크 · 오디언스 타겟팅 메뉴는 마스터 역할만 볼 수 있어요. 사용자·광고주 계정에는 해당 메뉴가 아예 나타나지 않아요.
2
STEP 2
매체 연동
🔌
광고 매체(네이버, 구글, 메타 등)의 API를 연결하면 그날그날 성과가 자동으로 모입니다. 매체마다 API 발급 방법이 다르니, 먼저 각 매체의 광고주(비즈니스) 계정을 준비해두세요.
🔑
① API 키 입력
설정 → 리포트 설정 → 매체 연동에서 네이버 검색광고 · 구글 Ads · 메타 · 카카오 등 매체별 API 키를 입력합니다. 매체 이름 옆 "API 키 설정" 버튼을 누르면 무엇을 입력해야 하는지 안내가 나와요.
✅
② 연동 상태 확인
같은 화면 상단의 매체 연동 상태 표에서 각 매체가 정상적으로 연결됐는지 한눈에 확인할 수 있어요. 빨간 표시가 있다면 API 키를 다시 확인해보세요.
🎯
③ 픽셀(전환 추적 코드) 등록
같은 화면에서 매체별 픽셀 ID를 등록하면, 사용자가 우리 사이트에서 구매·가입 같은 행동을 했을 때 어느 광고를 보고 왔는지 추적할 수 있어요.
3
STEP 3
전환 데이터 입력
🎯
"전환"이란 우리가 원하는 사용자 행동(구매, 회원가입, 상담신청 등)을 말해요. 전환을 어떻게 정의하느냐에 따라 ROAS(광고비 대비 매출 비율)·CPA(전환 1건당 비용) 같은 핵심 숫자가 달라지므로, 처음에 꼼꼼히 설정해두는 게 중요해요.
⚙️
① 전환 이벤트 등록
전환데이터 입력 → 전환설정에서 "구매", "회원가입" 같은 전환 이벤트를 추가합니다. 이벤트마다 데이터 출처(GA4 · MMP · 직접입력)를 선택해요.
🔗
② 외부 솔루션 연동 (GA4 등)
출처로 GA4나 MMP(Airbridge·Adjust·AppsFlyer 등)를 선택하면 연동 정보를 입력하는 창이 자동으로 열려요. 안내에 따라 속성 ID, API 키 등을 입력하면 연동이 완료됩니다.
📝
③ 수기 전환 데이터 입력
오프라인 상담·계약처럼 픽셀로 잡히지 않는 전환은 전환데이터 입력 → 수기 전환 데이터 입력에서 매체·지표별 표에 직접 숫자를 입력하면 됩니다. 조회 기간은 기본적으로 어제 하루로 설정되어 있고, 입력한 값은 대시보드 KPI에 바로 반영돼요.
4
STEP 4
리포트 설정
📐
매체마다 리포트에 쓰는 용어가 조금씩 달라요(예: 네이버는 노출수를 "impCnt"라고 써요). 이 용어를 DeepFle의 표준 이름으로 맞춰두면, 나중에 데이터를 붙여넣거나 다운로드할 때 자동으로 알아봅니다.
🗂️
① 컬럼 매핑
설정 → 리포트 설정 → 컬럼 매핑에서 매체별로 "원본 컬럼명(매체 리포트에 적힌 이름)"과 "솔루션 컬럼명(DeepFle 표준 이름)"을 짝지어 저장합니다. 이렇게 해두면 Raw 업로드에서 리포트를 그대로 붙여넣을 때 자동으로 인식돼요.
🔢
② 전환수는 외부 솔루션이 우선
GA4 같은 외부 솔루션이 연결된 매체는 그쪽 전환수가 자동으로 우선 사용돼요. 직접 입력한 값보다 외부 솔루션 값이 더 정확하다고 보기 때문이에요.
💹
③ 마크업(수수료) 설정
설정 → 리포트 설정 → 마크업에서 매체별 대행 수수료 비율(%)을 설정하면, 광고비에 자동으로 반영되어 클라이언트 보고서에 정확한 금액이 표시돼요.
5
STEP 5
목표 설정 및 알림
🔔
목표 숫자를 미리 입력해두면, 실제 성과가 목표에 못 미칠 때 바로 알 수 있어요.
📈
① KPI 목표 입력
설정 → 목표설정 및 알림 → KPI 목표에서 ROAS · CPA · CTR(클릭률) · CVR(전환율) 등 지표별 목표치를 매체별로 입력합니다. 대시보드 KPI 카드에 목표 달성 여부가 함께 표시돼요.
🔔
② 알림 채널 설정
설정 → 목표설정 및 알림 → 알림에서 KPI 이상 감지 · 예산 소진 · 주간 리포트 발송 알림을 항목별로 켜고 끌 수 있어요.
6
STEP 6
Raw 데이터 업로드신규
📤
API로 연결되지 않은 매체나, 아직 API 연동을 준비 중인 매체는 리포트를 직접 붙여넣어 데이터를 채울 수 있어요. Raw 업로드 메뉴에서 3가지 방법을 제공합니다.
🔌
① API 연결 매체
API로 이미 연동된 매체는 이 탭에서 수집 현황을 확인할 수 있어요. 별도 입력이 필요 없습니다.
📋
② 수동 업로드 (복사 붙여넣기)
매체 광고관리자에서 리포트를 복사한 뒤 그대로 붙여넣으면, STEP 4에서 설정한 컬럼 매핑에 따라 자동으로 항목이 분류돼요. 매칭되지 않은 컬럼은 그 자리에서 드롭다운으로 지정할 수 있고, 한 번 지정하면 다음부터는 자동으로 인식됩니다.
✏️
③ 수기 매체 등록 및 업로드
API 연동 자체가 없는 매체(오프라인 매체, 신규 플랫폼 등)는 매체명을 직접 등록한 뒤 데이터를 입력할 수 있어요. 등록한 매체는 이름 옆에 "(수기)" 표시가 붙은 채로 대시보드와 미디어 리포트에 자동으로 함께 집계됩니다.
7
STEP 7
대시보드 모니터링
📊
전체 매체 성과를 한 화면에서 확인하는 곳이에요. 매일 아침 여기부터 확인하는 걸 추천해요.
📅
① 조회 기간
기본 조회 기간은 어제 하루예요. 상단 날짜 선택기로 원하는 기간으로 바꿀 수 있고, "동기간 비교"를 체크하면 이전 기간과 자동으로 비교돼요.
🔄
② 전체 매체 동기화
화면 상단의 "전체 매체 동기화" 버튼을 누르면 연동된 모든 매체의 데이터를 그 자리에서 즉시 최신 상태로 갱신할 수 있어요.
⚠️
③ 이상 감지 배너
KPI 목표에서 벗어난 매체가 있으면 대시보드 상단에 배너로 알려줘요. 배너를 누르면 어떤 매체·지표에 문제가 있는지 바로 확인할 수 있어요.
💡
④ 인사이트
화면 하단에 [효율변화] · [인사이트] · [제안] 형식으로 AI 분석 결과가 표시될 자리가 준비되어 있어요. 실제 데이터를 바탕으로 한 AI 분석 기능은 곧 연결될 예정입니다.
8
STEP 8
미디어 리포트 활용
📈
매체 하나를 골라 더 자세히 들여다보고 싶을 때 사용하는 메뉴예요. 일별·주별·월별 추세를 함께 볼 수 있어요.
📡
① 매체 선택 후 조회
미디어 리포트에서 살펴볼 매체를 고르고 "조회"를 누르세요. 연동된 매체와 수기로 등록한 매체 모두 목록에 나타나요.
📱
② 디바이스별로 나눠보기
전체 / 모바일 / PC / 태블릿 필터로 기기별 성과를 따로 확인할 수 있어요. 모바일과 PC의 차이를 보고 소재나 입찰 전략을 조정해보세요.
📥
③ 엑셀 · PDF로 내보내기
화면 상단의 엑셀 · PDF 버튼으로 지금 조회한 리포트를 그대로 파일로 저장할 수 있어요. 클라이언트 보고용으로 바로 활용하세요.
9
STEP 9
Raw 데이터 다운로드
⬇️
가장 세부적인 원본 데이터가 필요할 때 사용하는 메뉴예요. 캠페인 · 광고그룹 · 키워드 · 소재 단위까지 내려받을 수 있어요.
🔍
① 원하는 조건 선택
Raw 다운로드에서 기간 · 매체 · 컬럼을 고릅니다. 필요한 컬럼만 선택하면 파일이 더 가벼워져요. 컬럼 순서는 Raw 업로드에서 설정한 순서를 그대로 따라가요.
🗂️
② 계층별로 쪼개보기
캠페인 · 광고그룹 · 키워드 · 소재 컬럼을 함께 켜면, 어떤 키워드나 소재가 잘 되고 안 되는지 한 파일에서 바로 비교할 수 있어요.
📊
③ 디바이스까지 함께 보기
"디바이스" 컬럼을 추가하면 캠페인별·기기별 성과를 교차로 볼 수 있어요. 소재 A/B 테스트 결과를 분석할 때 유용해요.
10
STEP 10
활동 피드 & 팀 메시지신규
📰
팀원이 계정 안에서 무엇을 했는지 확인하고, 같은 계정을 함께 담당하는 사람들끼리 대화도 나눌 수 있는 공간이에요.
🗒️
① 활동 기록 (자동)
활동 피드를 열면 매체 연동, 데이터 업로드, 설정 변경 같은 주요 작업이 "누가 · 언제 · 무엇을 했는지"와 함께 시간순으로 자동 기록돼요. 따로 입력할 필요 없이 계정 안에서 벌어진 일을 그대로 보여줍니다.
💬
② 팀 메시지
같은 계정을 담당하는 마스터·사용자끼리 화면 오른쪽에서 메시지를 주고받을 수 있어요. "@이름"을 입력하면 팀원을 멘션할 수 있고요. 광고주 역할에게는 이 영역이 보이지 않아요.
