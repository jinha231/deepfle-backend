STEP 1
계정 설정
🚀
DeepFle를 시작하려면 먼저 광고주 계정을 생성하고 팀 멤버를 초대하세요. 역할에 따라 접근 가능한 기능이 달라집니다.
🏢
계정 생성
관리자 로그인 후 설정 → 계정 관리에서 광고주(클라이언트)별 계정을 생성합니다. 계정당 하나의 브랜드 또는 광고주를 대응시킵니다.
👥
역할별 멤버 초대
관리자(MASTER) — API 연동·설정·사용자 관리 전체 권한
담당자(USER) — 성과 조회·분석·리포트 편집
광고주(Advertiser) — 성과 데이터 조회 전용 (편집 불가)
🔒
접근 권한 안내
자동 최적화·어트리뷰션 링크·오디언스 타겟팅 메뉴는 관리자(MASTER)만 접근 가능합니다. USER·광고주 계정은 해당 메뉴가 노출되지 않습니다.
2
STEP 2
매체 연동
🔌
광고 매체 API를 연결하면 성과 데이터가 자동으로 수집됩니다. 매체별 API 발급 절차가 다르므로 각 플랫폼의 비즈니스 계정을 준비하세요.
🔑
API 키 등록
설정 → 매체 연동에서 네이버SA · 구글 Ads · 메타 · 카카오 등 광고 플랫폼 API 키를 입력합니다. 각 매체의 "API 키 설정" 버튼을 클릭하면 필요한 항목이 안내됩니다.
🎯
픽셀 ID 등록
각 매체 픽셀 ID를 등록하여 랜딩 페이지 전환을 추적합니다. 픽셀이 설치된 URL에서 실시간 전환 데이터를 수집할 수 있습니다.
📊
외부 솔루션 연결
GA4 · AppsFlyer · Airbridge 등 어트리뷰션 솔루션을 추가로 연결합니다. 멀티 터치 어트리뷰션 분석 시 필수입니다.
3
STEP 3
전환 지표 설정
🎯
어떤 사용자 행동을 "전환"으로 볼지 정의합니다. 정확한 ROAS·CPA 측정의 기준이 됩니다.
⚙️
전환 이벤트 추가
리포트 설정 → 전환설정에서 구매 · 회원가입 · 상담신청 등 추적할 이벤트를 등록합니다. 이벤트명은 픽셀에 설정된 이벤트명과 일치해야 합니다.
💰
전환값 설정
이벤트별 가치(예: 구매 = 평균 주문금액)를 입력하면 ROAS가 자동 계산됩니다. 전환값이 없는 이벤트는 CPA 기준으로만 집계됩니다.
📝
수기 전환 보완
오프라인 성약 · 상담 완료 등 픽셀로 추적되지 않는 전환은 수기 전환 입력으로 보완합니다. 입력된 수기 전환은 대시보드 KPI에 자동 반영됩니다.
4
STEP 4
리포트 컬럼 설정
📐
연동된 매체의 API 필드를 DeepFle 지표 컬럼에 매핑합니다. 현재 계정에 연결된 매체만 목록에 표시되며, 매체별 API 문서를 참조하여 필드명을 입력하세요.
🗂️
필드 매핑 입력
설정 → 매체-솔루션간 컬럼 동기화에서 연동된 매체별로 노출수 · 클릭수 · 광고비 · 전환수 · 전환매출 API 필드명을 직접 입력합니다. 입력란에 각 매체 API 필드명 예시가 안내됩니다. 입력 후 저장 버튼을 눌러 적용하세요.
🔗
전환수 외부 솔루션 우선 적용
전환수 컬럼은 외부 어트리뷰션 솔루션(GA4 · AppsFlyer 등)이 연결된 경우 해당 솔루션 데이터가 자동으로 우선 적용됩니다. 직접 입력한 API 필드값보다 외부 솔루션 전환수가 우선됩니다.
💹
마크업 설정
설정 → 마크업에서 매체별 광고비 마크업 비율(%)을 설정합니다. 에이전시 수수료가 광고비에 자동 반영되어 클라이언트 보고서에 정확한 금액이 표시됩니다.
5
STEP 5
KPI 목표 & 알림 설정
🔔
목표 KPI를 입력하고 이탈 시 즉시 알림을 받아 빠르게 대응하세요.
📈
KPI 목표 입력
설정 → KPI 목표에서 ROAS · CPA · CTR · CVR 지표별 목표치를 입력합니다. 목표치는 대시보드 KPI 카드에 달성 여부와 함께 표시됩니다.
🔔
알림 채널 설정
설정 → 알림에서 이상 감지 알림을 받을 이메일 채널을 설정합니다. 규칙 실행 알림 · 주간 리포트 발송 등 항목별로 개별 활성화할 수 있습니다.
6
STEP 6
대시보드 모니터링
📊
전체 매체 성과를 한눈에 모니터링하고 기간 비교 분석을 활용하세요.
📅
기간 설정 및 비교
상단 날짜 선택기로 조회 기간을 설정합니다. 기본 조회 기간은 전일(어제) 기준 최근 30일로 설정됩니다. "동기간 비교" 체크 시 직전 동일 기간과 자동 비교되며 KPI 카드에 증감이 표시됩니다.
⚠️
이상 감지 배너
KPI 목표 이탈 시 대시보드 상단에 이상 감지 배너가 표시됩니다. 배너를 클릭하면 이상 발생 매체와 지표를 확인할 수 있습니다.
💡
인사이트 복사
하단 인사이트 섹션의 "복사" 버튼으로 분석 내용을 클립보드에 복사할 수 있습니다. 주간·월간 보고서 작성 시 바로 활용하세요.
7
STEP 7
미디어 리포트 활용
📈
매체별 상세 성과를 분석하고 일별·주별·월별 추세를 확인합니다.
📡
매체 선택 및 조회
미디어 리포트 메뉴에서 분석할 매체를 선택하고 조회 버튼을 누릅니다. 연동된 매체에 한해 실제 데이터가 표시됩니다.
📱
디바이스별 분석
디바이스 필터(전체 / 모바일 / PC / 태블릿)로 기기별 성과를 분리 확인합니다. 모바일과 PC의 성과 차이를 파악하여 소재·입찰 전략에 활용하세요.
📥
리포트 다운로드
엑셀 · PDF 버튼으로 현재 조회된 리포트를 내보냅니다. 클라이언트 보고용으로 활용하거나 내부 데이터 보관용으로 사용하세요.
8
STEP 8
Raw 데이터 다운로드
⬇️
원시 데이터를 캠페인·광고그룹·키워드·소재 단위로 세밀하게 분석하거나 외부 보고서 작성에 활용합니다.
🔍
조회 조건 설정
Raw 다운로드 메뉴에서 기간·매체·컬럼을 선택합니다. 필요한 항목만 선택하여 다운로드 파일 크기를 줄일 수 있습니다.
🗂️
계층 분석
캠페인 · 광고그룹 · 키워드 · 소재 컬럼을 활성화하면 세부 계층별 성과를 한 파일에서 분석할 수 있습니다. 비효율 키워드·소재 발굴에 유용합니다.
📊
디바이스 컬럼 추가
"디바이스" 컬럼을 함께 활성화하면 기기별·캠페인별 교차 성과 데이터도 다운로드할 수 있습니다. 소재 A/B 테스트 분석에 활용하세요.
